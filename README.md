# 🏗️ Construction Cost Tracker v1

A professional Microsoft Excel dashboard for tracking construction project costs.

## 📌 Features

- Track construction material purchases
- Automatic Total Cost calculation
- Dashboard KPIs
- Highest Purchase
- Average Purchase
- PivotTable summary
- PivotChart visualization
- Category Slicer
- Dynamic Excel Tables
- No VBA required
