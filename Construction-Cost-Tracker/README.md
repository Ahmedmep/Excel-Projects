# Construction Cost Tracker v1

## 📌 Overview
A professional Microsoft Excel dashboard for tracking construction project costs.

This project was built using Excel Tables, PivotTables, PivotCharts, Slicers, and modern dashboard design principles without using VBA.

## ✨ Features

- Track all purchases
- Automatic calculations
- Interactive Dashboard
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline Filter
- KPI Cards
- Excel Table (Dynamic)

## 🛠 Built With

- Microsoft Excel 2016
- Pivot Tables
- Pivot Charts
- Structured References
- Excel Tables
